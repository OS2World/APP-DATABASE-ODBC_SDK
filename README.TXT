* ODBC SDK README FOR OS/2

This file contains infromation about any changes made to the program after
the documentation was printed.

The updated DB2 ODBC files for OS/2 are located in the ODBC/DB2DRV directory 
on the installation CD.  Before you can use the DB2 ODBC Driver CAE/2 software 
must be installed.  Please refer to the DB2ODBC.TXT document located in the 
ODBC/DB2DRV directory for further information.
